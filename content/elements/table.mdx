## Table Element

The TableElement is a flexible component in TypeComposer, designed for creating dynamic and styled tables. It extends the functionality of the standard HTML table, allowing property-driven customization for headers, rows, and overall table styling.

## 1. Key Features

- Allows for dynamic configuration of headers and rows using properties.
- Supports customization of styles for table, headers, and cells.
- Provides integration with other TypeComposer components for flexible layouts.
- Includes responsive and interactive features, such as row highlighting.

## 2. Basic Example

```typescript
import { TableElement } from "typecomposer";

class ExampleTable extends TableElement {
	constructor() {
	super({
		headers: ["Name", "Age", "Occupation"],
		rows: [
		["Alice", "30", "Engineer"],
		["Bob", "25", "Designer"],
		["Charlie", "35", "Manager"],
		],
		className: "border border-gray-300",
	});
	}
}

export default ExampleTable;
```

## 3. Properties

| Property | Type | Description | Example |
|----------|------|-------------|---------|
| headers | string[] | An array of column headers for the table. | `["Name", "Age", "Occupation"]` |
| rows | string[][] | A 2D array representing the rows of the table. | `[["Alice", "30"], ["Bob", "25"]]` |
| className | string | CSS class for styling the table. | `"border border-gray-300"` |
| cellClassName | string | CSS class for styling individual cells. | `"px-4 py-2 text-left"` |
| striped | boolean | Enables striped rows for better readability. | `true` |
| hoverable | boolean | Adds hover effects to rows. | `true` |
| onclick | function | Event handler triggered on row clicks. | `(row) => console.log(row)` |

## 4. Examples

### Basic Table with Headers and Rows

```typescript
new TableElement({
	headers: ["Name", "Age", "City"],
	rows: [
	["John", "28", "New York"],
	["Jane", "32", "San Francisco"],
	["Sam", "24", "Los Angeles"],
	],
	className: "border border-gray-300",
});
```

### Striped and Hoverable Table

```typescript
new TableElement({
	headers: ["Product", "Price", "Stock"],
	rows: [
	["Laptop", "$1000", "Available"],
	["Smartphone", "$700", "Out of Stock"],
	["Tablet", "$500", "Limited"],
	],
	striped: true,
	hoverable: true,
	className: "border border-gray-300",
});
```

### Interactive Table with Row Click Event

```typescript
new TableElement({
	headers: ["Task", "Status"],
	rows: [
	["Design Homepage", "Completed"],
	["Implement API", "In Progress"],
	["Write Documentation", "Pending"],
	],
	onclick: (row) => console.log("Row clicked:", row),
	className: "border border-gray-300",
});
```

## 5. Default Behavior

By default, the TableElement:
- Displays a basic table without any additional styling.
- Does not apply row hover or striping unless explicitly enabled.
- Allows for dynamic header and row definitions through properties.

## 6. When to Use TableElement

- When you need to create dynamic, data-driven tables with minimal effort.
- For creating styled and interactive tables in your application.
- When you want to integrate tables with other TypeComposer components.

## Conclusion

The TableElement simplifies the process of creating structured, styled, and interactive tables in TypeComposer. Its property-driven approach allows for easy customization and integration, making it ideal for displaying tabular data in a clean and efficient way.
