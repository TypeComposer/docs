# Get Started with TypeComposer

Follow these steps to create a new TypeComposer project quickly.

## 1. Use the official project initializer

Run the following command:

```bash
npm create typecomposer@latest
```

Youâ€™ll be prompted to provide some basic information:

* **Template**: Project or Library
* **Project Name**
* **Package Name**
* **Router**: yes or no (optional)
* **TailwindCSS**: yes or no (optional)

## 2. Navigate and install dependencies

```bash
cd <project-name>
npm install
```

## 3. Start the development server

```bash
npm start
```

Your app will now be running locally. Open it in your browser and start building components!
